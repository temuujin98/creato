# creato

`creato` is the frontend foundation for a Mongolian-first AI creative studio.

Current scope:

- React
- TypeScript
- Vite
- Tailwind CSS
- Static production-safe homepage
- Static placeholder data for future `Category -> Product` workflows

Out of scope for the current phase:

- Backend
- Auth
- Admin panel
- Supabase
- Database
- Wallet
- Payment
- AI generation
- API routes

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Brand Notes

- Visible brand name: `creato`
