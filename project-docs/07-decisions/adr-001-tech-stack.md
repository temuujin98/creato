# ADR-001: Tech Stack

## Status

Accepted

## Decision

Use React, TypeScript, Vite, and Tailwind CSS for the initial frontend foundation.

## Context

The project needs a clean, fast frontend setup with strong typing and a scalable component model.

## Consequences

- Fast local development.
- Simple static deployment path.
- Clear future path for frontend expansion.
- Backend and operational systems remain separate future work.
