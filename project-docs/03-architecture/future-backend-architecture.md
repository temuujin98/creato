# Future Backend Architecture

No backend is implemented in Phase 1.

Future backend responsibilities may include:

- Authentication and account management.
- Category and product APIs.
- Credit wallet and ledger.
- Payment verification.
- Upload handling.
- Generation job orchestration.
- AI provider abstraction.
- Admin-only prompt and provider configuration.

## Boundary

Prompt templates, provider credentials, generation instructions, and admin logic must remain server-side.
