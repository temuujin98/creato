# Roadmap

## Phase 0: Foundation

- Initialize Vite React TypeScript app.
- Configure Tailwind CSS.
- Create project documentation.
- Establish scalable folders.

## Phase 1: Homepage

- Create public homepage.
- Add static content, category data, product data, and pricing data.
- Keep all operational systems unimplemented.

## Future Phase 2: Product Discovery

- Product listing page.
- Category detail pages.
- Product detail UI.
- Static option schemas.

## Future Phase 3: Auth and Accounts

- User auth.
- Session handling.
- Protected app shell.

## Future Phase 4: Credits and Payment

- Wallet.
- Credit purchase flow.
- Payment provider integration.
- Ledger and transaction history.

## Future Phase 5: Generation

- Upload flow.
- Backend generation jobs.
- AI provider abstraction.
- Result gallery.

## Future Phase 6: Admin

- Admin panel.
- Category and product management.
- Pricing management.
- Prompt and provider configuration on the server side only.
