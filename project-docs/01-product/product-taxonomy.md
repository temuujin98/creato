# Product Taxonomy

`creato` uses a `Category -> Product` model.

## Category

Categories group business use cases.

Initial placeholder categories:

- E-commerce
- Restaurant
- Fashion
- Beauty

## Product

Products are specific creative workflows inside categories.

Initial placeholder products:

- Product photo
- Campaign poster
- Menu visual
- Social post

## Future Product Fields

- Required image count.
- Optional image count.
- Option schema.
- Default credit cost.
- Output aspect ratios.
- Prompt template reference stored server-side.
- Availability status.

Prompt templates and AI provider instructions must not be exposed to the client.
