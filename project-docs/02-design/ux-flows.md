# UX Flows

## Phase 1 Homepage Flow

1. User lands on hero.
2. User understands `creato` as an AI creative studio.
3. User reviews value proposition.
4. User explores categories and products.
5. User learns the future workflow.
6. User reviews pricing.
7. User reaches FAQ or final CTA.

## Future Generation Flow

1. User logs in.
2. User selects category.
3. User selects product.
4. User uploads required images.
5. User selects product options.
6. User confirms credit usage.
7. Backend creates generation job.
8. User views generated result.

This future flow is documented but not implemented in Phase 1.
