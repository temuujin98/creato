# Design System

## Visual Direction

- Dark mode first.
- Premium black and white SaaS mood.
- Cinematic abstract lighting.
- Restrained accent colors.
- Clear spacing and readable typography.

## Typography

Use a Mongolian-friendly system stack:

```text
Inter, Noto Sans, Noto Sans Mongolian, ui-sans-serif, system-ui, sans-serif
```

## UI Principles

- Keep the first screen focused on the core promise.
- Avoid pricing in the hero.
- Use clear, compact sections.
- Make mobile layout first-class.
- Avoid one-file UI implementations.

## Brand

- Visible brand name: `creato`
- Do not use `Creato.mn` as the visible brand name.
- Do not add punctuation to the wordmark.
