# Homepage Spec

## Goal

Create a premium dark homepage for `creato`, a Mongolian-first AI creative studio.

## Sections

- Hero
- Value proposition
- Featured categories
- How it works
- Featured products
- Why creato
- Credit packages / Pricing
- Showcase
- FAQ
- Final CTA
